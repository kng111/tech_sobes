Python интерпритируемый 

Копилируемый язык - сложный в разработки считается на машинном коде (или типо того)



pep8 - PEP8 — это официальное руководство по стилю кода Python. PEP расшифровывается как P ython E nhancement P roposal («Предложение по улучшению Python»). Это документы, предлагающие новые особенности языка. Они образуют официальную документацию особенности языка, принятие или отклонение которой обсуждается в сообществе Python.



Интерпретируемый язык-это тип языка программирования,
 который не компилируется в машинный код,
 а интерпретируется компьютером во время выполнения и при необходимости переводится в машинно понятный код.



Python 3.7 (и 3.8):
Data Classes: Введены "data classes" для более удобного создания классов, предназначенных преимущественно для хранения данных.
Type Hints: Улучшения в подсказках типов (Type Hints), включая поддержку типов данных "Tuple" и "Generic".
Context Variables: Появились контекстные переменные (Context Variables).

Python 3.8:
Assignment Expressions: Введен новый оператор := (walrus operator), что упрощает присвоение в выражениях.
Positional-Only Parameters: Появилась поддержка параметров только для позиционных аргументов в сигнатуре функции.
f-strings с "=": Добавлена возможность использовать = в f-строках для отображения выражений и их значений.
Python 3.9:

New Syntax Features: Новые синтаксические возможности, такие как оператор | для соединения словарей и новые методы строк.
Type Hinting Improvements: Дополнительные возможности и улучшения в подсказках типов.
Zoneinfo: Добавлен новый модуль zoneinfo для более удобной работы с временными зонами.

Python 3.10:
Pattern Matching: Введено структурное сопоставление шаблонов для более удобной работы с данными.
Precise Types: Появились новые типы, такие как zoneinfo.ZoneInfo.
Parenthesized Context Managers: Появилась поддержка использования нескольких менеджеров контекста в одном выражении с использованием круглых скобок.
